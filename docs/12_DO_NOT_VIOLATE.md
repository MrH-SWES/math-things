# Do Not Violate

Hard rules for all agents working in this repository.

---

- **Do not turn apps into lessons.** The app is the object. Lesson framing belongs outside the app.
- **Do not add visible instructions as a substitute for better interaction.** If the interface isn't obvious, fix the interface.
- **Do not add dashboards, badges, points, levels, or reward systems.**
- **Do not make math objects look like cheap plastic, candy, jelly, flat UI, or clip art.**
- **Do not let the equation drive the object.** Records follow action. Action does not follow records.
- **Do not edit multiple apps at once unless explicitly instructed.**
- **Do not move root HTML files unless explicitly instructed.**
- **Do not break GitHub Pages URLs.**
- **Do not add frameworks, bundlers, or package managers unless explicitly instructed.**
- **Do not add Tool Chest or Dimensional Builder active-project context to this repo yet.**
